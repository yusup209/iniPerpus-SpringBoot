# iniPerpus

Aplikasi ini merupakan sistem manajemen perpustakaan yang ringkas, mudah digunakan, dan cepat. Menggunakan framework Spring Boot. Untuk bahasa pemrograman yang digunakan adalah Java dan Anda bisa mengembangkannya di NetBeans.

### Prasyarat Tools untuk Pengembangan
- NetBeans terbaru
- Java 25 (JDK)
